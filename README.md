# MiniProject1

This code takes a set of sales data, and cleans it as preprocessing.
Specifically, the main changes are removals of non-purchases from the data to evelaute the true sales data for the calendar year.
Each segment is title-commented to explain what the block is doing. 
TO RUN, RESTART THE RUNTIME, THEN RUN ALL BLOCKS

The report of the findings and decisions log are listed as pdf files.
The dataset comes from Kaggle: https://www.kaggle.com/datasets/carrie1/ecommerce-data

The most significant findings:
November is the most profitable month in terms of revenue. The top 10 items in terms of revenue do not completely match the top 10 items in terms of quantity sold. Regional analysis supports that the most profitable regions are the UK, and regions nearest the UK. International buyers typically buy more in wholesale bulk. These bulk orders are reflected by 31.99% of the revenue sourced from the top 10 buying customers.
